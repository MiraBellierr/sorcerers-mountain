A Platform game written with html and Javascript using canvas tag by me.

## How to play
* **Hold I** - To see instruction
* **Press S** - To start a game
* **Arrow Up** or **Spacebar** - Jump
* **Arrow Left** - Go left
* **Arrow Right** - Go right
